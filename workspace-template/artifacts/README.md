# Общие артефакты

Используйте этот каталог только для межделовых, воспроизводимых и разрешенных результатов. Материалы конкретного дела обычно хранятся в `matters/<id>/outputs/`.

Каждый artifact должен иметь:

- source/matter reference;
- generated_at и tool/version;
- validation/test report;
- classification;
- human review status;
- checksum для опубликованной версии.
