# Глоссарий

| Термин | Рабочее значение |
|---|---|
| Claim | Существенный тезис итогового текста с уникальным ID |
| Evidence unit | Точное адресуемое предложение/абзац/ячейка источника |
| Evidence ledger | Таблица claim → supporting/counter evidence |
| Source register | Реестр происхождения, версий, статусов и хэшей источников |
| As-of date | Абсолютная дата, на которую проверяется право/факт |
| Output | Непосредственный продукт деятельности |
| Outcome | Изменение поведения/доступности/качества практики |
| Impact | Изменение состояния общественной проблемы |
| Evidence gate | Минимальный набор проверок/источников до drafting |
| Handoff | Краткая передача устойчивого состояния следующей сессии |
| ADR | Запись устойчивого архитектурного/процессного решения |
