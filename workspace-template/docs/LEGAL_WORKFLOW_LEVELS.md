# Уровни юридической работы L0-L3

## L0 — проверенный точечный ответ

Подходит, когда вопрос узкий, риск ограничен, факты ясны и достаточно 1-3 первичных источников.

Минимум: вопрос/as-of, краткий вывод, exact authorities, ограничения. Source register может быть компактным.

Эскалировать, если есть конфликт редакций, спорная практика, персональные последствия или недостаток фактов.

## L1 — юридическая записка/opinion

Подходит для решения конкретного правового вопроса и рекомендации действия.

Минимум: MATTER/PLAN, source register, issue analysis, adverse authority, claim ledger, human review. Структура: вопрос → краткий ответ → право → анализ → риски → вывод/действие.

Эскалировать, если нужно делать вывод о системной практике, статистике или общественном воздействии.

## L2 — исследовательское досье

Подходит для сложного/спорного вопроса, большого массива практики, сравнительного анализа или подготовки policy options.

Минимум: воспроизводимые search protocols/corpora, evidence ledger, conflicts, missing data, options matrix, independent audit.

Эскалировать, если задача спрашивает результативность регулирования, причинность, root cause и reform package.

## L3 — правовой мониторинг

Подходит для оценки нормы/института как инструмента государственной политики.

Минимум: monitoring passport, hypotheses, theory of change, normative/practice/data modules, causal matrix, root-cause tree, ranked recommendations, evidence pack и audit.

## Матрица выбора

| Признак | L0 | L1 | L2 | L3 |
|---|---:|---:|---:|---:|
| Узкий вопрос/мало источников | ✓ | ✓ | | |
| Юридически значимое решение | | ✓ | ✓ | ✓ |
| Спорная/массовая практика | | | ✓ | ✓ |
| Количественные данные | | optional | ✓ | ✓ |
| Общественная проблема/outcome/impact | | | optional | ✓ |
| Причинность/counterfactual | | | optional | ✓ |
| Проект реформы/systemic recommendations | | | ✓ | ✓ |

Выбирай минимальный уровень, достаточный для риска и решения. Нельзя маскировать недостаток доказательств большим объемом текста.
